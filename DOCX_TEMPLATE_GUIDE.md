# Руководство по созданию DOCX шаблона карты

## Как создать шаблон

1. Открой Word и создай новый документ
2. Настрой размер страницы: 540px x 320px (или 14.3cm x 8.5cm)
3. Создай дизайн обратной стороны карты

## Плейсхолдеры для данных

Используй следующие плейсхолдеры в тексте (они будут заменены на реальные данные):

### Группа крови
- `{{bloodTypeFull}}` - полная группа крови (например: "II Rh+")
- `{{bloodType}}` - только группа (например: "II")
- `{{rhFactor}}` - только резус (например: "Rh+")

### Аллергии
Для списка аллергий используй цикл:
```
{#allergies}
• {{name}}
{/allergies}
```

Или просто одну строку для каждой:
- `{{allergies[0].name}}` - первая аллергия
- `{{allergies[1].name}}` - вторая аллергия
- `{{allergies[2].name}}` - третья аллергия

### Хронические заболевания
Для списка заболеваний используй цикл:
```
{#diseases}
• {{name}}
{/diseases}
```

Или просто одну строку для каждой:
- `{{diseases[0].name}}` - первое заболевание
- `{{diseases[1].name}}` - второе заболевание
- `{{diseases[2].name}}` - третье заболевание

### QR код
- `{{qrUrl}}` - URL для QR кода (можно вставить как текст или использовать для генерации QR)

## Пример структуры

```
┌─────────────────────────────────────────────┐
│  СКАНИРУЙТЕ ТЕЛЕФОНОМ    ГРУППА КРОВИ       │
│                                              │
│     [QR КОД]             {{bloodTypeFull}}  │
│                                              │
│                          АЛЛЕРГИИ            │
│                          {#allergies}        │
│                          • {{name}}          │
│                          {/allergies}        │
│                                              │
│                          ХРОНИЧЕСКИЕ         │
│                          ЗАБОЛЕВАНИЯ         │
│                          {#diseases}         │
│                          • {{name}}          │
│                          {/diseases}         │
│                                              │
│  [Синяя полоска с текстом внизу справа]     │
└─────────────────────────────────────────────┘
```

## Сохранение

Сохрани файл как `backend/templates/card-back.docx`

## Примечания

- Можешь использовать любое форматирование (цвета, шрифты, размеры)
- Плейсхолдеры должны быть в фигурных скобках: `{{название}}`
- Для циклов используй `{#массив}` и `{/массив}`
- QR код пока вставляется как текст URL (можно будет добавить генерацию изображения позже)
